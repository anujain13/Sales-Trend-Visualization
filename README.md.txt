# 📊 Sales Trend Visualization and Prediction

## Overview

This project analyzes retail sales data using the Sample Superstore dataset to uncover business insights, identify sales patterns, evaluate customer behavior, and predict sales performance using machine learning techniques.

The project covers the complete Data Analytics workflow including:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Sales Trend Analysis
* Machine Learning Prediction Models
* Interactive Streamlit Dashboard

---

## Project Objectives

* Analyze historical sales performance
* Identify regional and category-wise trends
* Study customer segment behavior
* Visualize key business metrics
* Build predictive models for sales classification
* Develop an interactive dashboard for decision-making

---

## Dataset

**Dataset Used:** Sample Superstore Dataset

The dataset contains retail transaction information such as:

* Order Date
* Ship Date
* Sales
* Profit
* Quantity
* Discount
* Region
* Category
* Customer Segment
* Sub-Category
